--- SETUP VIA RUNCOMMAND (portal UI, no SSH needed) ---

Step 1: Go to ReverseEngineeringVM → Operations → Run command → RunShellScript

Step 2: Paste this into the script box and click Run:

apt update
apt install -y docker.io docker-compose-v2
git clone https://github.com/sundhar-04/Reverse-Engineering-Contest.git
cd Reverse-Engineering-Contest
echo "JWT_SECRET=$(openssl rand -hex 64)" > .env
docker compose up -d

Step 3: Open ports
VM → Networking → Add inbound port rules:
  - Port 3000, Priority 1010, Name: frontend
  - Port 8000, Priority 1020, Name: backend

Step 4: Go to VM → Overview → copy Public IP → open http://<IP>:3000



--- STOP / START / DELETE ---
# Stop VM (saves $):
Go to VM → Overview → Stop (or az vm deallocate)

# Start VM:
VM → Overview → Start

# Delete everything:
Resource groups → ReverseCodeRG → Delete resource group
