--- PORTAL VM CREATION STEPS ---

1. portal.azure.com → Virtual machines → Create → Azure virtual machine

2. BASICS tab:
   Subscription:   Azure for Students
   Resource group: Create new → "ReverseCodeRG"
   VM name:        "ReverseCodeVM"
   Region:         Pick one that's NOT grayed out (try West US, West US 2, etc.)
   Image:          "Ubuntu Server 22.04 LTS - x64 Gen2"
   Size:           Click "See all sizes" → search "B2s" → select "Standard_B2s"
   Username:       "azureuser"
   SSH public key source: "Generate new key pair"
   Key pair name:  "ReverseCodeVM_key"
   → Click "Next: Disks >"

3. DISKS tab:
   (defaults are fine) → "Next: Networking >"

4. NETWORKING tab:
   (defaults are fine) → "Review + create"

5. Click "Create"
   → Download the .pem file when prompted
   → Wait for deployment to finish

6. Go to resource → copy the Public IP address

--- SETUP VIA CLOUD SHELL ---

# Upload deploy script to VM
scp -i ~/Downloads/ReverseCodeVM_key.pem deploy-azure.sh azureuser@<IP>:~

# SSH in and run setup script (clones repo, starts Docker, creates admin)
ssh -i ~/Downloads/ReverseCodeVM_key.pem azureuser@<IP> 'bash deploy-azure.sh'

# Or do it step by step:
ssh -i ~/Downloads/ReverseCodeVM_key.pem azureuser@<IP>
# Then run:
sudo apt update && sudo apt install -y docker.io docker-compose-v2
git clone https://github.com/sundhar-04/Reverse-Engineering-Contest.git
cd Reverse-Engineering-Contest
echo "JWT_SECRET=$(openssl rand -hex 64)" > .env
sudo docker compose up -d
